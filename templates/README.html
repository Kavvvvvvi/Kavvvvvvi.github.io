博客格式模板库

这个文件夹包含了可在 Markdown 博客文章中使用的各种 HTML/CSS 格式模板。
使用这些模板可以让您的博客文章更加美观和结构化。

可用模板

  1. 引言框模板 - 用于突出显示文章引言或重要概述
  2. 警告/提示框模板 - 用于显示提示、警告、成功或错误信息
  3. 代码块带标题模板 - 用于显示带有标题的代码片段
  4. 引用模板 - 用于美化引用内容
  5. 并排图片模板 - 用于并排显示多张图片

使用方法

  1. 打开您想要使用的模板文件
  2. 复制HTML代码
  3. 粘贴到您的Markdown博客文章中
  4. 替换示例内容为您自己的内容

自定义

每个模板文件中都包含了注释，说明如何自定义颜色、大小等样式。
您可以根据自己的喜好修改这些样式。
